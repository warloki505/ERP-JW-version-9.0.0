TESTES MANUAIS RECOMENDADOS (cenários críticos de ponta a ponta)
================================================================================

TESTE A — Ciclo completo online
  1. Logar no Device A e Device B com a mesma conta
  2. No Device A: criar receita R$1.000, despesa R$300, excluir a despesa
  3. Aguardar até 10s
  4. No Device B: verificar que receita aparece, despesa NÃO aparece
  5. KPIs de ambos os devices devem estar idênticos

TESTE B — Conflito de edição simultânea
  1. Device B: colocar offline
  2. Device A: editar TX-001, descrição = "Versão A", salvar (sync imediato)
  3. Device B (offline): editar TX-001, descrição = "Versão B", salvar (fica na fila)
     *** Device B deve ter updatedAt POSTERIOR ao Device A ***
  4. Device B: reconectar
  5. Resultado esperado: TX-001 com descrição "Versão B" em ambos os devices

TESTE C — Ressurreição de dado excluído
  1. Device B: colocar offline
  2. Device A: excluir TX-002, aguardar sync
  3. Device B (offline): TX-002 ainda aparece (ainda não recebeu o delete)
  4. Device B: reconectar
  5. Resultado esperado: TX-002 desaparece do Device B (soft delete prevalece)

TESTE D — Bootstrap completo
  1. Device B: abrir aba anônima (localStorage vazio)
  2. Logar com a mesma conta do Device A
  3. Aguardar bootstrap (badge: "Executando bootstrap..." → "Sync: OK")
  4. Verificar: transações, categorias, bancos e metas idênticos ao Device A

TESTE E — Restore de backup cross-device
  1. Device A: exportar backup JSON
  2. Device B (aba anônima): logar, localStorage vazio
  3. Device A: importar backup JSON em Perfil
  4. Aguardar sync
  5. Device B: dados do backup devem aparecer sem ação manual
